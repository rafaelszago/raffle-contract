# <img src="logo.svg" alt="RaffleLand" height="40px">

RaffleLand is a Web3 Application to create raffles in blockchain.

## Features

- truffle
- eslint
- solhint
- solidity-coverage
- faker
- @openzeppelin/test-helpers

## Overview

### Installation

Install the project dependencies.

`yarn install`

### Build

Build the project for production.

`yarn build`

### Test

Test the project.

`yarn test`

### Lint

Lint the code.

``yarn lint``
